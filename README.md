## 🇺🇸 English

**Litematica Auto Ignite** is a client-side Minecraft mod that automatically ignites Nether portals based on Litematica schematics.

### ✨ Features

- **Automatic Ignition**: Detects obsidian frames in the schematic and automatically uses a flint and steel.
- **Litematica Integration**: Works directly with your loaded schematics.
- **Safety**: Client-side only operation.

### 📥 Installation

1. Download and install [Fabric Loader](https://fabricmc.net/use/).
2. Download [Fabric API](https://modrinth.com/mod/fabric-api).
3. Download [Litematica](https://www.curseforge.com/minecraft/mc-mods/litematica) and [MaLiLib](https://www.curseforge.com/minecraft/mc-mods/malilib).
4. Download **Litematica Auto Ignite**.
5. Place all `.jar` files into your `mods` folder.

## 🇷🇺 Русский

**Litematica Auto Ignite** - это клиентский мод для Minecraft, который автоматически зажигает порталы в Нижний мир, основываясь на схематиках Litematica.

### ✨ Особенности

- **Автоматическое зажигание**: Мод обнаруживает блоки обсидиана в схематике и автоматически использует зажигалку.
- **Интеграция с Litematica**: Работает напрямую с вашими загруженными схематиками.
- **Безопасность**: Работает на стороне клиента.

### 📥 Установка

1. Скачайте и установите [Fabric Loader](https://fabricmc.net/use/).
2. Скачайте [Fabric API](https://modrinth.com/mod/fabric-api).
3. Скачайте [Litematica](https://www.curseforge.com/minecraft/mc-mods/litematica) и [MaLiLib](https://www.curseforge.com/minecraft/mc-mods/malilib).
4. Скачайте **Litematica Auto Ignite**.
5. Поместите все `.jar` файлы в папку `mods`.
